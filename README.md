# FotoBeam

App macOS nativa SwiftUI per caricare album batch su Google Photos.

## Avvio

```bash
swift run FotoBeam
```

## File locali

- `credentials.json`: credenziali OAuth Google. Copia `credentials.example.json` e inserisci i tuoi valori locali.
- `token_swift.json`: token OAuth salvato localmente dall'app.
- `report_upload_swift.json`: report di upload e ripresa generato dall'app.

Questi file sono esclusi da git per evitare di pubblicare credenziali, token OAuth, percorsi locali o nomi di file personali.
